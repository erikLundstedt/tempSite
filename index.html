how to install a modpack in multimc


start the program multimc
if you dont already have it
you can download it from this link. you are also going to need a zipfile
containing the modpack.
here is a link to a coupple of packs i have made.


the next step is as easy as dragging and dropping the .zip file to the multiMC window.
shown in the image is the edit window.
here you can change the name ,group and icon of the pack.
you also get here if you press the "add instance" button in the multiMC UI.
when you are happy with the name and icon you can press OK.

when you have pressed "ok" your computer might freeze for anything between a second to a coupple of minutes.
when this is done, you will see a screen simular to this one.
! by default multiMC will have a white/light theme !
you might not see the cute cat when you first install it,
to get the cute cat to show up
you will need to press the black icon next to the patreon donate button

in order to run the pack you will need to loggin
you do this by pressing the icon with the text "profiles" and shosing "manage accounts".
here you have to loggin with your minecraft account.

all of my packs uses the java path found in a linux based OS.
this means that you will have to go to the settings section under "edit instance".
now check the "java installation" box and click auto detect.
one of the alternatives should have a yellow star beside it
this is the one you should probably chose.

if your game crashes it MIGHT be because
you havent given java enough memory
to fix this you will have to check the "memory" checkbox.
and increase the amount of "maximum memory alocation" to about doubble (4096 MB)
